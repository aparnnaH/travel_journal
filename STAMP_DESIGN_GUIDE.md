# Premium Passport Stamp Design Guide

## Overview
All 10 country stamps have been completely redesigned with a significantly improved vintage passport aesthetic, premium collectible feeling, and enhanced visual detail. Each stamp maintains the 160x160 viewBox format optimized for 90x90px display size.

---

## Design Philosophy

### Visual Enhancements
- **Premium Vintage Aesthetic**: Decorative borders, ornamental frames, and classical proportions
- **Texture & Depth**: Layered gradients, opacity variations, and pattern overlays
- **Cultural Authenticity**: Accurate representations of iconic cultural elements
- **Collectible Quality**: Decorative seals, country codes, and premium finishes
- **Ink Stamp Feel**: Varied stroke widths, opacity layers, and textured backgrounds

### Color Strategy
Each stamp uses a **primary color**, **secondary color**, and **background** to create visual harmony:
- Primary colors drive the main design elements
- Secondary colors add accent details and decorative flourishes
- Backgrounds employ subtle gradients and textures for premium feel

## Technical Implementation

### SVG Features Across All Stamps

1. **Gradients**:
   - Radial gradients for glowing effects (suns, halos)
   - Linear gradients for backgrounds and bands
   - Gradient color transitions for premium feel

2. **Patterns**:
   - Paper/linen textures as backgrounds
   - Decorative pattern overlays
   - Subtle opacity variations

3. **Layering**:
   - Multiple opacity levels (0.3 to 1.0)
   - Overlapping elements for depth
   - Background-to-foreground hierarchy

4. **Details**:
   - Varied stroke widths (0.5px to 3px)
   - Decorative borders and frames
   - Corner ornaments and flourishes
   - Collectible seals with country codes

5. **Vintage Elements**:
   - Dashed borders for classic stamp appearance
   - Text seals at bottom (country codes)
   - Paper texture backgrounds
   - Aged color palettes
